The Challenge

The Institute of Applied AI at MSU managed years of farm research data stored in raw text files. Every reporting cycle, their team manually imported this data into Excel, built charts and tables by hand, and assembled PDF reports. A process that consumed an entire week of work each time.

The process was fragile, time-intensive, and left no room for on-the-fly analysis or flexibility.

Our Approach

Moore AI built a full-stack web application using Python and Django that ingests the raw data files automatically, eliminating all manual importing. The platform puts the research team in control: build custom charts, define their own formulas, and generate polished PDF reports in a single click.

What We Delivered

Automated Data Pipeline

Raw .txt files ingested and processed automatically. No manual imports.

Interactive Dashboard

Live visualizations of farm data with flexible chart and table creation.

Custom Formula Builder

Users define their own formulas to generate new metrics and charts on demand.

One-Click PDF Reports

Professional, formatted reports generated instantly from the platform.

User Management

Role-based access so the right people see the right data.

Full Web Application

Accessible from any browser. No software installs or Excel required.
